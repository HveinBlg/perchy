==========================================
    Perchy 桌面宠物 (macOS) - 快速上手
==========================================

一只坐在你窗口标题栏上、跟着窗口走的小猫咪。
每 3 分钟随机换一只姿势，一共 9 种。

------------------------------------------
一、怎么打开
------------------------------------------
1. 把这个文件夹整个解压到你想要的位置（比如"应用程序"或"下载"）
2. 双击  Perchy.app

首次运行如果 macOS 弹出 "无法打开 Perchy，因为它来自身份不明的开发者":
   - 右键 Perchy.app → 打开
   - 弹窗上再点  "打开"

或者:
   - 系统设置 → 隐私与安全性
   - 拉到底部，会看到 "已阻止使用 Perchy"
   - 点  "仍要打开"


------------------------------------------
二、给"辅助功能"权限 (必须做!)
------------------------------------------
Perchy 需要读取其他应用窗口的位置，所以必须授权:

1. 打开  系统设置 → 隐私与安全性 → 辅助功能
2. 找到 Perchy (如果没有就点 + 号加进去)
3. 打开右边的开关

授权后 Perchy 会自动跟随激活的窗口。
不授权就看不到猫。


------------------------------------------
三、看不到猫？
------------------------------------------
- 点一下任何窗口（Safari、浏览器、访达都行）
- 猫会出现在那个窗口的标题栏上，跟着窗口走
- 如果焦点在桌面（没有窗口被选中），猫会自动隐藏

如果所有窗口点了都不出猫:
- 检查上面第二步的"辅助功能"授权
- 或者退出 Perchy 重新打开


------------------------------------------
四、想开机自动出猫
------------------------------------------
1. 系统设置 → 通用 → 登录项
2. 点"打开时"下面的 +
3. 选择  Perchy.app
4. 下次开机自动出猫 🐱

不想开机自启：在同样位置删掉即可。


------------------------------------------
五、想换成自己的猫（或者其他形象）
------------------------------------------
把你的 PNG 图片（透明背景最好）
丢进 Perchy.app 旁边的  assets/pets  文件夹

想删掉某只猫，直接删对应的 png 就行。
程序每 10 秒自动扫这个文件夹，不用重启。

单张图就是一只"宠物"。放几张就会在这几张里随机切。


------------------------------------------
六、怎么关掉
------------------------------------------
Perchy 是"后台应用"，没有 Dock 图标。想关掉:

- 打开  活动监视器  (启动台里找)
- 搜索  Perchy
- 选中它，点左上角 X 号退出


------------------------------------------
七、目前 macOS 版没有的功能
------------------------------------------
- 最大化窗口时的"点鞭炮还原": macOS 没有 Windows 那样的
  "向下还原"按钮，先不做。Windows 版有这个。

其他所有功能都跟 Windows 版一致。


------------------------------------------
八、遇到问题
------------------------------------------
到 GitHub 提 Issue:
