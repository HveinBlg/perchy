# 推广物料

这个文件夹里都是**给项目维护者用的营销物料**，不面向终端用户。

## 目录

- **[小红书脚本.md](小红书脚本.md)** — 3 条 15-30 秒短视频完整分镜、
  BGM 推荐、封面设计建议、发布时段与限流词避坑
- **[V2EX发帖.md](V2EX发帖.md)** — 3 个标题候选（附 A/B 建议）、
  完整帖子正文、常见评论的回复模板、发布运营节奏

## 使用节奏（推荐）

### 周六
- **早上 9-11 点**：发小红书视频 1（治愈款）
- **下午**：手机拍屏、剪映拼视频（30 分钟）
- **晚上 21-22 点**：发小红书视频 2（秀技术款）

### 周日
- **晚上 21-22 点**：发小红书视频 3（萌宠款）
- **晚上 21:30-22:30**：同时段发 V2EX 帖子

### 48 小时后
- 数据填到脚本文件末尾的表格里
- 拿数据说话决定下一步（继续小红书 / 转向掘金 / 停手当作品放着）

## 关键指标

小红书：**收藏率**（收藏 / 播放）比点赞率更能预测下载
V2EX：**帖子下的 GitHub 链接点击率** 比 Star 数更能预测真实兴趣
