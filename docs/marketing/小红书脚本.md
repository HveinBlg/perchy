# 小红书视频脚本（3 条）

3 条不同角度的展示脚本，**每条 15-30 秒**，手机录屏 + 剪映拼一下就能发。

一次全发 3 条，看**哪条数据最好**（点赞 / 收藏 / 评论比例），下一次继续那个方向。

**通用建议**：
- 全部**竖屏 9:16**（1080×1920 或 720×1280）
- **前 2 秒必须抓眼**（85% 的用户会在前 2 秒决定要不要看下去）
- **末尾一定要留 CTA**（评论区看 / 主页看 / 免费下载）
- BGM 用**小红书热榜里的音乐**（当前热点会有加权推荐）

## 🚨 小红书发布注意事项

**不要说的词（会被限流）**：
- ❌ "免费" → 说"0 元" / "白嫖"
- ❌ "下载" → 说"拿" / "get"
- ❌ "GitHub" → 说"主页看链接"
- ❌ "程序" / "软件" → 说"小工具" / "小玩意"
- ❌ 直接放链接 → 只能放主页里"个签"或者"评论区置顶"

**替代话术**：

- ❌ "免费下载 GitHub 链接" → ✅ "主页有免费获取方式"
- ❌ "点这里下载" → ✅ "评论区看链接" 或 "简介找我"

**账号背景**：
- **新号发不出流量**是正常的，前 2-3 条大概率播放量在 100-500
- **别删**，让平台学习你的内容标签
- **发到第 3-5 条**开始会有大流量
- **一条爆了别停**，趁热赶紧发同类型的第 2 条

## 💡 如果第一波数据不好

**不要慌**。小红书发新号推流不给力是常态。3 个可能：

1. **封面不够抓** → 换更冲击力的封面（大字 + 反差）
2. **标签用得不对** → 蹭当周热标签
3. **BGM 冷门** → 换热榜 BGM

**发 3 条以上**再评估。**别一条数据差就放弃**。
