==========================================
    Perchy - 桌面宠物快速上手
==========================================
具体介绍：https://vpsbt.online/archives/586/

一只坐在你窗口标题栏上、跟着窗口走的小猫咪。
每 3 分钟随机换一只姿势，一共 9 种。

英文版本请看同目录下的 USAGE.txt。


==========================================
          Windows 用户看这里
==========================================


一、怎么打开
------------------------------------------
1. 把这个文件夹整个解压到你想要的位置（比如桌面）
2. 双击  perchy.exe

首次运行如果 Windows 弹出 "Windows 已保护你的电脑"：
   - 点 "更多信息"
   - 点 "仍要运行"

（这是因为 exe 没花钱买签名证书，不是病毒。
  源码开源在 https://github.com/HveinBlg/perchy 可以看。）


二、看不到猫？
------------------------------------------
点一下任何窗口（浏览器、微信、文件管理器都行），
猫会出现在那个窗口的标题栏上；窗口一动，猫跟着走。

如果焦点在桌面（没有窗口被选中），猫会自动躲起来。


三、想开机自动出猫
------------------------------------------
1. 按 Win + R
2. 输入 shell:startup 回车
3. 把 perchy.exe 的快捷方式拖进那个文件夹
   （右键 perchy.exe -> 发送到 -> 桌面快捷方式，
     然后把桌面这个快捷方式挪进 shell:startup 文件夹）
4. 下次开机自动出猫

不想开机自启：删掉 shell:startup 里那个快捷方式即可。


四、想换成自己的猫（或其他形象）
------------------------------------------
1. 把你的 PNG 图片（透明背景最好）
   丢进 assets\pets 这个文件夹
2. 想删掉某只，直接删对应的 png 就行
3. 程序每 10 秒自动扫这个文件夹，不用重启

单张图就是一只 "宠物"。放几张就会在这几张里随机切。


五、怎么关掉
------------------------------------------
- 双击  stop.bat
- 或者按 Ctrl + Shift + Esc 打开任务管理器，
  找到 perchy.exe，右键 -> 结束任务



==========================================
           macOS 用户看这里
==========================================


零、先确认你下的是对的版本
------------------------------------------
Perchy 有两个 macOS 版：

  Perchy-vX.Y.Z-macos.dmg          -> 现代版 (macOS 11 及以上)
  Perchy-vX.Y.Z-macos-legacy.dmg   -> 旧版兼容 (macOS 10.15 Catalina)

查你 Mac 的系统版本：苹果 -> 关于本机

  你的系统             下哪个
  ------------------   ---------------------------
  15  Sequoia          现代版 (-macos.dmg)
  14  Sonoma           现代版
  13  Ventura          现代版
  12  Monterey         现代版
  11  Big Sur          现代版（最低要求）
  10.15 Catalina       旧版兼容 (-macos-legacy.dmg)
  10.14 及更老          不支持，请升级系统

两个版本功能完全一样，只是内部用的 Qt / Python 版本不同。
Catalina 用户装现代版会看到 "此应用程序要求 macOS 11 或更高版本"，
那种情况就下 -legacy.dmg。


一、怎么打开 (dmg 版)
------------------------------------------
1. 双击下载好的 Perchy-vX.Y.Z-macos.dmg
   -> 会弹出一个磁盘窗口，看到：
        Perchy.app
        assets   (文件夹)
        Applications  (快捷方式)
        使用说明.txt / USAGE.txt

2. 把 Perchy.app 和 assets 文件夹一起拖到同一个位置。
   两种做法二选一：

   做法 A（推荐，简单）：
     全部拖到 "应用程序" 或 桌面
     -> 就是把 Perchy.app 和 assets 拖到同一个文件夹里

   做法 B：
     把 Perchy.app 拖到 "应用程序" 快捷方式上
     把 assets 拖到 ~/Applications/（跟 Perchy.app 一个文件夹）

   注意：assets 必须跟 Perchy.app 在同一个文件夹，否则猫图找不到。

3. 双击 Perchy.app

首次运行如果弹 "无法打开 Perchy 因为 Apple 无法检查其中是否有恶意软件"：
   - 右键 Perchy.app -> 打开
   - 弹窗上再点 "打开"

或者：
   - 系统设置 -> 隐私与安全性
   - 拉到底部，会看到 "已阻止使用 Perchy"
   - 点 "仍要打开"


二、给"辅助功能"权限（必须做！）
------------------------------------------
Perchy 需要读其他应用窗口的位置，所以必须授权：

1. 打开 系统设置 -> 隐私与安全性 -> 辅助功能
2. 找到 Perchy（如果没有就点 + 号加进去）
3. 打开右边的开关

授权后 Perchy 会自动跟随激活的窗口。
不授权就看不到猫。


三、看不到猫？
------------------------------------------
- 点一下任何窗口（Safari、浏览器、访达都行）
- 猫会出现在那个窗口的标题栏上，跟着窗口走
- 如果焦点在桌面（没有窗口被选中），猫会自动隐藏

如果所有窗口点了都不出猫：
- 检查上面第二步的 "辅助功能" 授权
- 确认 assets 文件夹在 Perchy.app 旁边（不在 assets 就没图可显示）
- 退出 Perchy 重新打开


四、想开机自动出猫
------------------------------------------
1. 系统设置 -> 通用 -> 登录项
2. 点 "打开时" 下面的 +
3. 选择 Perchy.app
4. 下次开机自动出猫

不想开机自启：在同样位置删掉即可。


五、想换成自己的猫（或其他形象）
------------------------------------------
把你的 PNG 图片（透明背景最好）
丢进 Perchy.app 旁边的 assets/pets/ 文件夹

想删掉某只，直接删对应的 png 就行。
程序每 10 秒自动扫这个文件夹，不用重启。

单张图就是一只 "宠物"。放几张就会在这几张里随机切。


六、怎么关掉
------------------------------------------
Perchy 是后台应用，没有 Dock 图标。想关掉：

- 打开 活动监视器（启动台里搜）
- 搜索 Perchy
- 选中它，点左上角 X 号退出


七、目前 macOS 版没有的功能
------------------------------------------
"最大化窗口时的点鞭炮还原" 是 Windows 独有：macOS 没有 Windows
那种明确的 "最大化" 状态，所以还没做。

其他所有功能都跟 Windows 版一致。
